# AI 助手记忆系统

## 测试流程（必须记住）

### 微信小程序测试流程
1. **先在 Onboarding 填写信息** → 点击生成穿搭
2. **生成完成后**，点击右上角「设置」图标
3. **再次进入 Onboarding 页面**，此时才会显示「返回」按钮
4. **首次进入**（无历史数据）不会显示返回按钮

### 发布流程（必须按顺序）
1. 修改代码
2. 自测（运行 test.sh）
3. 部署（node deploy.js）
4. 生成预览（node preview.js）
5. 发送二维码给用户确认

## 常见问题修复

### 返回按钮不显示
- 原因：dashboard.js 中使用了 reLaunch 而不是 navigateTo
- 解决：使用 navigateTo 跳转，保留页面栈

### 返回点击无反应
- 解决：增加 fallback 逻辑，如果页面栈为空则用 reLaunch
