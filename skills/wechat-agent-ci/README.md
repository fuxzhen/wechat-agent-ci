# 🚀 WeChat-Agent-CI: 一键部署微信小程序的 AI 脚手架

> V3.0 升级：引入了产品经理思维，Agent 在撸起袖子写代码前，会先给你发 PRD 和 UI 线框图，你点头了它才干活。测试驱动，安全可靠。

## 🛠️ 快速上手
1. 复制 `.env.example` 为 `.env` 并填入 AppID。放置 `private.key` 到 frontend 目录。
2. `cd frontend && npm install`

## 🧙‍♂️ AI Agent 食用指南
让你的 Agent 阅读 `OPENCLAW_SKILL.md`。
现在你可以直接对它下达一句话需求，它会严格遵循：**输出可视化方案 -> 等你确认 -> 写测试 -> 写代码 -> 自测闭环 -> 扫码预览** 的完美工作流。

最后，在根目录执行 `git add .` 和 `git commit -m "feat: 升级 V3.0，引入可视化需求评审与确认机制"`。
